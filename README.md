
# Paxi

Paxi is a landlord assistance that will help keep track of every property, 
tenant and payment you may have. 
## Tech Stack

**Framework:** UIKit

**Local Persistance:** CoreData

**Mayor UIComponents:** CollectionViews, TableViews, FSCalendar
